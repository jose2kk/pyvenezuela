# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.0.5] - 2026-05-06

### Added

- `py.typed` marker for PEP 561 inline type support
- Python 3.14 to test matrix and classifiers
- CHANGELOG.md following Keep a Changelog format
- CONTRIBUTING.md, CODE_OF_CONDUCT.md, and SECURITY.md
- GitHub issue templates (bug report, feature request) and PR template
- Comprehensive tests for BCV client, cache, and decorators (95% coverage)

### Changed

- Migrated from Dependabot to Renovate for dependency management

## [0.0.4] - 2026-04-20

### Changed

- Migrated build system to uv
- Added ty for type checking
- Added mkdocs for documentation
- Updated CI workflows

## [0.0.3] - 2024-11-11

### Added

- In-memory cache implementation for API responses

## [0.0.2] - 2024-11-11

### Changed

- Updated BCV client
- Updated README

## [0.0.1] - 2024-05-26

### Added

- Initial release
- CNE client for querying cedula data
- BCV client for querying exchange rates
- PyPI package publishing
